新的一年，定几个目标，白纸黑字写下来，方便年底复盘。

## 学习目标

### 深入一门语言

今年想把 **Go** 语言学扎实。之前只是浅尝，写了一些脚本，今年目标是：

- [ ] 读完《Go 程序设计语言》
- [ ] 用 Go 写一个完整的小项目（CLI 工具或 HTTP 服务）
- [ ] 理解 goroutine 和 channel 的底层原理

### 系统学习计算机基础

- [ ] 复习操作系统核心概念（进程、内存管理、文件系统）
- [ ] 刷完 100 道 LeetCode 题（重点：树、图、动态规划）
- [ ] 了解 TCP/IP 协议栈

### 前端技能提升

- [ ] 系统学习 TypeScript
- [ ] 深入了解 Vue 3 Composition API

## 阅读计划

今年想读完这几本书：

1. 《Go 程序设计语言》— Alan Donovan
2. 《深入理解计算机系统》（CSAPP）— 经典中的经典
3. 《原则》— Ray Dalio
4. 《被讨厌的勇气》— 岸见一郎

## 输出计划

- 每月至少写 **2 篇博客**
- 至少完成 **1 个开源小项目**

## 写在最后

计划不是用来完美执行的，是用来提供方向的。

> 重要的不是计划本身，而是计划的过程——它让你清楚地想明白自己想要什么。

期待年底复盘的时候，能在这些方框里打上勾 ✅
